- Spring Boot 2.7 框架
- MySQL 数据库（8.x 版本）
- Elastic Stack

  - Kibana 数据可视化
- 数据抓取
  - 离线
  - 实时
- 设计模式 
  - 门面模式
  - 适配器模式
  - 注册器模式
- 数据同步
  - 定时
  - 双写
  - Logstash
  - Canal
- JMeter 压力测试
